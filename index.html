Elephant Framework

Web Application Framework in Python and Javascript. Made for ERPNext

Introduction

Elephant Framework (formerly wnframework) is a full stack application framework built to make database driven applications like ERPs. This framework was built for ERPNext and can be extended to make other applications.

This framework is large and complex and might take some time to understand.
